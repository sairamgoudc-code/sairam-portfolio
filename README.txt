Sai Ram - Portfolio (Vite)

Files included:
- src/ (React source, App.jsx)
- public/profile.jpg (placeholder - replace with your photo if desired)
- public/Sai_Ram_IT_Operations_Engineer_Resume.pdf (your resume) (FOUND & COPIED)

How to use:
1. Unzip the project and `cd` into the folder.
2. Run `npm install` to install dependencies.
3. Start dev server: `npm run dev`
4. Build: `npm run build`
5. For Vercel: push to GitHub and import the repo in Vercel (it auto-detects Vite).

Notes:
- Tailwind is included via dev dependencies; ensure you run `npm install` before dev server.
- If your real profile photo wasn't uploaded, replace public/profile.jpg with your image named exactly 'profile.jpg' before deploying.
